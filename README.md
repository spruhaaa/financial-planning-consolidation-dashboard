# financial-planning-consolidation-dashboard

This project simulates an enterprise financial planning and consolidation system 
similar to SAP BPC using Excel and Power BI.

✅ Features ->
- Financial Data Modeling (Fact & Dimension Tables)
- Budget Planning Templates (Excel input forms)
- Currency Conversion (FX Rate logic)
- Intercompany Elimination
- Financial Consolidation (Group-level reporting)
- Power BI Dashboard (Interactive P&L)

---

📈 Dashboard Includes ->
- Profit & Loss (P&L) Statement
- Revenue, COGS, Net Income
- Entity-wise Analysis
- Slicers (Month, Entity, Version)
- Trend Charts

---

🛠 Tools Used ->
- Excel (Data + Planning)
- Power BI (Visualization)
- DAX (Measures & Calculations)

---

🧠 Key Learnings ->
- Star Schema Data Modeling
- Financial Consolidation Logic
- Power BI Reporting & DAX
- Enterprise-level BI Architecture

---

🚀 Future Enhancements ->
- Budget vs Actual Comparison
- Variance Analysis
- Drill-down reports
- Scenario Simulation
